# 🤖 AI/ML Engineering Internship Tasks – DevelopersHub Corporation

This repository contains my completed tasks for the **AI/ML Engineering Internship at DevelopersHub Corporation**. The projects focus on practical applications of machine learning, data analysis, and large language models (LLMs).

---

## 📌 Overview

The goal of this repository is to build strong hands-on experience in core AI/ML concepts by working on real datasets and real-world problems.

It includes:
- Data exploration and visualization
- Regression and classification models
- Time series prediction
- Prompt engineering with LLMs
- Chatbot development

---

## 📁 Tasks Included

### 📊 Task 1: Data Exploration & Visualization
- Dataset: Iris Dataset
- Performed data analysis using Pandas
- Visualized distributions and relationships using Matplotlib & Seaborn

### 📈 Task 2: Stock Price Prediction
- Dataset: Yahoo Finance (yfinance)
- Built regression model to predict next-day closing price
- Compared actual vs predicted prices

### ❤️ Task 3: Heart Disease Prediction
- Dataset: UCI Heart Disease Dataset
- Built classification models (Logistic Regression / Decision Tree)
- Evaluated using accuracy, confusion matrix, and ROC curve

### 🩺 Task 4: Health Query Chatbot (LLM-based)
- Used OpenAI / Hugging Face models
- Applied prompt engineering for safe and clear responses
- Built a simple conversational assistant

### 💬 Task 5: Mental Health Chatbot (Fine-Tuned)
- Dataset: EmpatheticDialogues
- Fine-tuned transformer model for empathetic responses
- Focused on supportive and safe AI conversations

### 🏠 Task 6: House Price Prediction
- Dataset: Kaggle House Prices
- Regression model using ML algorithms
- Evaluated using MAE and RMSE
